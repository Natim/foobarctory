foobar'ctory
============

Install
-------

Make sure you are using Python 3.6+

.. code-block:: bash

   python3.7 -m venv .venv
   source .venv/bin/activate

.. code-block:: bash

   pip install -e .

Run
---

.. code-block:: bash

   foobarctory

Show help
---------

.. code-block:: bash

   foobarctory --help
